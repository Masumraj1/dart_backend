# flutter_backend

১. MongoDB Tap অ্যাড করুন
brew tap mongodb/brew

২. এবার MongoDB ইনস্টল করুন
brew install mongodb-community@7.0

৩. সার্ভিস স্টার্ট করুন
brew services start mongodb-community@7.0

৪. এখন আপনার ডার্ট সার্ভার রান করুন
dart run lib/bin/server.dart

* সার্ভার বন্ধ করতে চাইলে: টার্মিনালে Control + C চাপুন।
